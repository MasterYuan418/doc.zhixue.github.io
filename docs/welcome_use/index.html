欢迎使用！

zhixue-middleware是一个基于zhixuewang-python的智学网中间件。它能与在zhixuewang-python下编写的代码无缝工作。

免责声明：本项目所有代码均遵守MPL 2.0协议开源。获取的信息皆为学习交流使用，不作其他任何用途，请在24小时内删除获取的信息。如果侵犯了您的权利，请联系我，我将尽可能维护您的权益。

感谢zhixuewang-python的作者 @anwenhu，他是本项目的前身的作者。

与zhixuewang-python的兼容性：

她与zhixuewang-python的代码比较如下：

zhixuewang-python代码

from zhixuewang import login
user = login("user","password")
user.get_self_mark()

zhixue-middleware代码

from zhixuewang import login
user = login("user","password")
user.get_self_mark()

可以看到，两个项目的代码是十分相像的，甚至可以通用:)

开始吧！​
