快速开始

目标：在5分钟内快速使用zhixue-middleware

提示：你也可以看看Github的自述文件

安装​

本项目是基于Python的，所以你需要先安装Python环境

前置条件​

  • Python 3.6.2~3.9.2
    • 3.6.2是最低测试版本，理论上3.X均可，但是没有经过测试。3.9.2是测试过的最高版本，理论上3.9以后也能支持，但也没有测试。
  • 手
    • 有手就行！

命令行安装​

打开命令提示符（或bash），输入如下指令：

pip install zhixue-middleware

这将会自动安装本项目，待pip完成后安装便完成。

手动安装​

caution

该方法不推荐在可以使用pip安装时使用！你永远要相信自动安装。

以下是命令：

git clone https://github.com/MasterYuan418/zhixue-middleware
cd zhixue-middleware
python setup.py install

获得成绩​

确保你已经安装zhixue-middleware库，之后运行如下程序：

from zhixuewang import login
user = login("账号","密码")
user.get_self_mark()

现在，你的成绩将会自动打印在屏幕上。

完成！​

本项目的快速使用教程到这里就结束了，你可以继续向下阅读，或离开本页面。
