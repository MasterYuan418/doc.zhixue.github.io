快速开始

目标：在5分钟内快速使用zhixuewang

提示：你也可以看看Github的自述文件

安装​

本项目是基于Python的，所以你需要先安装Python环境

前置条件​

  • Python 3.6.2~3.9.2
    • 3.6.2是最低测试版本，理论上3.X均可，但是没有经过测试。3.9.2是测试过的最高版本，理论上3.9以后也能支持，但也没有测试。
  • 手
    • 有手就行！

命令行安装​

打开命令提示符（或bash），输入如下指令：

pip install zhixuewang

这将会自动安装本项目，待pip完成后安装便完成。

手动安装​

caution

该方法不推荐在可以使用pip安装时使用！你永远要相信自动安装。

以下是命令：

git clone https://github.com/anwenhu/zhixuewang-python
cd zhuxuewang-python
python setup.py install

获得成绩​

确保你已经安装zhixuewang库，之后运行如下程序：

from zhixuewang import login
user = login("账号","密码")
data = user.get_self_mark()

此时你将会获得最近一次考试的成绩数据，你可以这样解析：

subjname = data[0].subject.name #获得第0项的学科名
score = str(data[0].score) #获得第0项的学科分数
code = str(data[0].subject.code) #获得第0项的学科代码

之后，你可以这样做：

print(subjname + " " + score + " " + code)

它将会输出类似于如下的内容：

语文 60 01

完成！​

本项目的快速使用教程到这里就结束了，你可以继续向下阅读，或离开本页面。
